# Assets-mapp

Denna mapp innehåller bilder och andra resurser för rapporten.

## rapporthuvud.svg

Placera din egen sidhuvudbild här. Bild ska vara:
- Format: SVG, PNG eller JPG
- Bredd: Minst 1200px för bra kvalitet
- Höjd: Cirka 200-300px (5cm i rapporten)

### Ersätta bilden

1. Ta bort eller ersätt `rapporthuvud.svg`
2. Uppdatera referensen i `rapport_mall.qmd`:

```markdown
<img src="assets/din-bild.svg" alt="Rapporthuvud" style="width: 100%; height: 5cm;"/>
```

### Tips

- SVG-format rekommenderas för vektorbaserade logotyper
- PNG för foton eller komplexa bilder
- Håll filstorleken under 1 MB för snabbare laddning
